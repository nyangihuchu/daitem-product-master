# 다이템 프로듀스 MVP PRD

## 🎯 핵심 정보
**목적**: 온라인 마켓터가 공급처와 상품 데이터를 한 곳에서 체계적으로 관리하고, 마진 계산 및 재고 상태를 자동화하여 의사결정 속도를 높인다.
**사용자**: 쿠팡·스마트스토어 등 여러 플랫폼에 상품을 판매하는 1인 또는 소규모 온라인 마켓터

---

## 🚶 사용자 여정

```
1. 서비스 진입
   ↓ 첫 방문 (미인증)
   로그인 페이지
   ↓ 회원가입 또는 로그인 성공

2. 대시보드 페이지 (메인 허브)
   ↓ 공급처가 0개인 경우 → "공급처를 먼저 등록하세요" 안내 배너 표시
   ↓ 상품이 0개인 경우 → "상품을 등록하세요" 안내 배너 표시
   ↓ 재고 주의 상품 존재 시 → 재고 주의 알림 카드 표시
   ↓ 사이드바 메뉴 선택
   ├── [공급처 관리] 선택 → 3단계
   ├── [상품 관리] 선택 → 4단계
   └── [재고 현황] 선택 → 5단계

3. 공급처 관리 페이지
   ↓ "공급처 등록" 버튼 클릭
   공급처 등록 폼 (슬라이드 패널)
   ↓ 저장 완료
   공급처 카드 목록에 반영
   ↓ 카드 클릭 → 수정/삭제 가능

4. 상품 관리 페이지
   ↓ "상품 등록" 버튼 클릭
   상품 등록 폼 (공급처 선택 필수)
   ↓ 매입가·판매가 입력 시 → 마진액·마진율 실시간 자동 계산
   ↓ 현재 재고 <= 최소 재고 시 → 상태 "재고부족" 자동 설정
   ↓ 저장 완료
   상품 목록 테이블에 반영
   ↓ 행 클릭 → 상품 수정/삭제 가능

5. 재고 현황 페이지
   ↓ 재고 수준 바 그래프 확인 (부족 상품 상단 정렬)
   ↓ 특정 상품 클릭 → 상품 관리 페이지의 해당 상품 수정 폼으로 이동
   ↓ 재고 수량 직접 수정 후 저장 → 재고 현황 실시간 반영
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|--------------|------------|
| **F001** | 공급처 CRUD | 국내/해외 구분, 업체명·담당자·연락처·결제조건·메모 등록·수정·삭제 | 상품과 공급처를 연결하기 위한 기반 데이터 | 공급처 관리 페이지 |
| **F002** | 공급처 카드형 목록 | 공급처를 카드 UI로 표시, 국내/해외 탭 필터 | 다수 공급처를 빠르게 파악하기 위한 핵심 UX | 공급처 관리 페이지 |
| **F003** | 상품 CRUD | SKU·상품명·카테고리·공급처·매입가·판매가·재고·플랫폼·상태 등록·수정·삭제 | 서비스의 핵심 데이터 자산 | 상품 관리 페이지 |
| **F004** | 마진 자동 계산 | 매입가·판매가 입력 시 마진액·마진율 실시간 자동 계산 (등록 폼 내) | 마켓터의 핵심 의사결정 지표 자동화 | 상품 관리 페이지 |
| **F005** | 재고부족 상태 자동 전환 | 현재 재고 <= 최소 재고 시 상품 상태를 "재고부족"으로 자동 변경 | 수동 상태 관리 오류 방지, 재고 리스크 자동화 | 상품 관리 페이지, 재고 현황 페이지 |
| **F006** | 상품 목록 및 필터 | SKU·상품명 검색, 공급처·상태·플랫폼 필터, 테이블형 목록 | 다수 상품을 빠르게 탐색하기 위한 핵심 조작 | 상품 관리 페이지 |
| **F007** | 재고 현황 시각화 | 전체 상품 재고 수준을 바 그래프로 표시, 재고부족 상품 우선 정렬 | 재고 리스크를 한눈에 파악하기 위한 핵심 뷰 | 재고 현황 페이지 |
| **F008** | 재고 수량 직접 수정 | 재고 현황 페이지에서 상품 재고 수량을 바로 수정 | 재고 업데이트 동선 최소화 | 재고 현황 페이지 |
| **F009** | 대시보드 통계 요약 | 전체 상품 수·공급처 수·재고부족 상품 수·평균 마진율 카드형 표시 | 서비스 전체 현황을 즉시 파악 | 대시보드 페이지 |
| **F010** | 재고 주의 알림 | 대시보드에서 재고부족 상품 목록을 카드형으로 표시, 해당 상품으로 바로 이동 | 긴급 재고 대응을 위한 핵심 알림 허브 | 대시보드 페이지 |

### 2. MVP 필수 지원 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|--------------|------------|
| **F011** | 기본 인증 | 회원가입·로그인·로그아웃 (Supabase Auth) | 개인 데이터 보호를 위한 최소 인증 | 로그인 페이지, 회원가입 페이지 |
| **F012** | 공급처-상품 연결 | 상품 등록 시 공급처를 드롭다운으로 선택, 연결 관계 저장 | 상품과 공급처 데이터 정합성 유지 | 상품 관리 페이지 |

### 3. MVP 이후 기능 (제외)

- 프로필 상세 관리 (아바타, 자기소개 등)
- 알림 설정 (이메일·푸시 알림)
- 상품 일괄 등록 (CSV 가져오기)
- 판매 채널 API 연동 (쿠팡, 스마트스토어)
- 주문·정산·배송 관리
- 팀 멤버 초대 및 권한 관리
- 상품 이미지 업로드
- 매입/판매 이력 추적
- 고급 분석 리포트 (기간별 마진 추이)

| **다음 이동** | 등록/수정 성공 → 카드 목록 즉시 반영, 삭제 → 해당 카드 제거 |

### 상품 관리 페이지
> **구현 기능:** F003, F004, F005, F006, F012 | **메뉴 위치:** 사이드바 "상품 관리"

| 항목 | 내용 |
|------|------|
| **역할** | 전체 상품 데이터를 테이블로 관리하고, 마진 자동 계산·재고부족 자동 전환이 이루어지는 핵심 페이지 |
| **진입 경로** | 사이드바 "상품 관리" 클릭, 대시보드 재고부족 카드 클릭, 재고 현황 페이지 상품명 클릭 |
| **사용자 행동** | 검색·필터로 상품 탐색, "상품 등록" 버튼으로 등록 폼 열기, 테이블 행 클릭으로 수정·삭제, 매입가·판매가 입력 시 마진 실시간 확인 |
| **주요 기능** | • 상품 테이블 목록 (SKU·상품명·공급처·매입가·판매가·마진율·재고·상태·플랫폼 컬럼)<br>• SKU·상품명 키워드 검색<br>• 공급처·상태·플랫폼 드롭다운 필터<br>• 상품 등록/수정 폼 (슬라이드 패널): 매입가·판매가 입력 시 마진액·마진율 실시간 자동 계산, 공급처 드롭다운 선택, 현재 재고 <= 최소 재고 시 상태 자동 "재고부족" 설정, 플랫폼 다중 태그 선택 (쿠팡·스마트스토어·11번가 등)<br>• 상품 삭제 (행 내 버튼) |
| **다음 이동** | 등록/수정 성공 → 테이블 즉시 반영, 삭제 → 해당 행 제거 |

### 재고 현황 페이지
> **구현 기능:** F005, F007, F008 | **메뉴 위치:** 사이드바 "재고 현황"

| 항목 | 내용 |
|------|------|
| **역할** | 전체 상품의 재고 수준을 바 그래프로 시각화하고, 재고부족 상품을 우선 확인·수정하는 전용 뷰 |
| **진입 경로** | 사이드바 "재고 현황" 클릭 |
| **사용자 행동** | 바 그래프로 재고 수준 한눈에 파악, 재고부족 상품 우선 확인, 인라인 재고 수량 수정 후 저장 |
| **주요 기능** | • 상품별 재고 수준 수평 바 그래프 (현재 재고 / 최소 재고 기준으로 색상 구분: 정상·주의·부족)<br>• 재고부족 상품 상단 우선 정렬<br>• 각 항목에서 재고 수량 인라인 직접 수정 후 저장<br>• 재고 수정 저장 시 재고부족 상태 자동 재계산·반영<br>• 상품명 클릭 시 상품 관리 페이지 해당 상품으로 이동 |
| **다음 이동** | 재고 수량 저장 → 현황 즉시 갱신, 상품명 클릭 → 상품 관리 페이지 |
